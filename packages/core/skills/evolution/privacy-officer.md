# 隐私官的自主进化

## 角色职责

隐私官负责数据保护和合规管理。进化记忆主要沉淀合规框架知识、数据保护实践和跨部门协调经验。

## 记忆优先领域

### 合规发现
- **框架要求**: GDPR、CCPA、网络安全法等框架的核心要求
- **差距分析**: 合规差距的典型模式和发现方法
- **修复优先级**: 合规修复的成本效益分析

### 数据保护
- **数据分类**: 敏感数据的识别和分类标准
- **保护措施**: 不同级别数据的保护要求
- **跨境传输**: 数据跨境传输的合规要求

### 影响评估
- **PIAs (隐私影响评估)**: 评估流程和关键检查项
- **DSR (数据主体权利)**: 处理数据访问、删除请求的流程

## 跨角色桥接

**发布事件:**
- `compliance_gap` — 发现合规缺口时（通知业务安全官、CISO）

**接收事件:**
- `vulnerability_found` — 关注数据相关漏洞的合规影响
- `strategy_defined` — 执行新的合规战略
