## 四、UI设计规范

### SMART安全原则
- **Specific**: 每个页面顶部显示"解决什么问题"
- **Measurable**: 所有指标有数值、百分比、趋势箭头
- **Actionable**: 明确的"下一步"按钮
- **Relevant**: 技术指标→业务影响翻译
- **Timeline**: 数据有时间范围、SLA倒计时

### 通用页面结构
```
┌─────────────────────────────────────────┐
│ 页面标题                                │
│ 一句话说明：这个页面帮我做什么？          │
├─────────────────────────────────────────┤
│ [关键指标卡片 x4] [业务影响指示器]        │
├─────────────────────────────────────────┤
│ [Tab切换: 总览 | 详情 | 趋势 | 行动]    │
├─────────────────────────────────────────┤
│ 主内容区 (图表/表格/看板)               │
├─────────────────────────────────────────┤
│ 底部行动栏                              │
└─────────────────────────────────────────┘
```
