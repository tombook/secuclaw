# SecuClaw 角色指挥台 UI 重设计

**日期：** 2026-04-11  
**状态：** 已批准  
**版本：** 1.0

---

## 1. 设计目标

将 SecuClaw UI 从传统的"安全仪表盘 + 8 个 Agent 标签"重新设计为**角色指挥台模式**，体现 20 年顶级安全专家经验的 AI 化身，而非简单的 Agent 选择器。

### 核心设计原则

1. **专家化身，非 Agent** - UI 传递的是"顶级安全专家随时待命"的感觉，而非"选择不同的 AI 工具"
2. **场景驱动** - 基于实时安全态势推荐相关角色和场景
3. **三者融合** - 每个角色指挥台融合 RACI 任务、专业指标、协作信息
4. **统一入口** - 保持全局安全总览为统一入口，角色视角为叠加层

## 2. 架构设计

### 2.1 整体页面结构

```
┌─────────────────────────────────────────────────────────────────┐
│  🔍 智能推荐条 (AI-driven Role & Scenario Recommendations)      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                          │   │
│  │              统一安全总览 / 角色指挥台                     │   │
│  │                                                          │   │
│  │   (根据当前上下文显示：总览 或 角色指挥台)                 │   │
│  │                                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 两种视图模式

| 模式 | 描述 | 入口 |
|------|------|------|
| **统一总览** | 全域安全态势仪表盘，展示所有角色的聚合数据 | 默认视图 |
| **角色指挥台** | 特定角色的专属视角（三者融合） | 通过智能推荐条或切换器进入 |

### 2.3 智能推荐条

**位置：** 页面顶部固定显示

**功能：**
- 根据实时安全态势自动推荐相关角色
- 根据当前场景推荐适合的操作
- 一键切换到推荐角色的指挥台

**AI 推荐逻辑：**
- 检测到高危漏洞 → 推荐 `security-expert`
- 检测到大规模安全事件 → 推荐 `secuclaw-commander`
- 合规审计周期 → 推荐 `privacy-officer`
- 供应链相关告警 → 推荐 `supply-chain-security`

## 4. 角色指挥台 - 三者融合

### 4.1 RACI 任务区

**设计：**
- 四个并列卡片：R（执行）、A（审批）、C（咨询）、I（通知）
- 每个卡片显示该类型任务数量和最新任务预览
- 点击卡片展开完整任务列表
- 支持拖拽改变任务状态

**交互：**
- R 任务 → 执行后自动通知相关角色
- A 任务 → 审批面板快速通过/驳回
- C 任务 → 跳转咨询详情页面
- I 任务 → 标记已读

### 4.2 专业指标区

**设计：**
- 网格布局的仪表盘卡片
- 每个角色 4-6 个核心 KPI
- 支持数据下钻（点击查看详情）
- AI 驱动的异常自动高亮

**数据联动：**
- 角色指挥台的数据与统一总览同步
- 任何视图的修改实时反映到所有相关界面

### 4.3 协作指挥区

**设计：**
- 当前战况讨论区
- 跨角色协调请求
- 专家会诊邀请
- 任务协作面板

**功能：**
- @提及特定角色请求支持
- 场景化讨论线程
- 协作历史追溯

## 5. 智能推荐条详细设计

### 5.1 推荐逻辑

```typescript
interface Recommendation {
  type: 'role' | 'scenario';
  priority: 'high' | 'medium' | 'low';
  title: string;
  description: string;
  action: () => void;
}

// 推荐触发条件示例
- 高危漏洞发现 → 推荐 security-expert + "漏洞响应" 场景
- 大规模安全事件 → 推荐 secuclaw-commander + "危机管理" 场景
- 合规审计周期临近 → 推荐 privacy-officer + "合规审计" 场景
- 供应链告警 → 推荐 supply-chain-security + "供应链审查" 场景
- 预算规划周期 → 推荐 ciso + "战略规划" 场景
```

### 5.2 视觉设计

- 高度：48px
- 背景：根据推荐强度渐变（高=红色系，中=橙色系，低=蓝色系）
- 显示：1-3 个推荐项，超出折叠
- 交互：点击推荐项 → 显示详情 + 快速操作按钮
- 关闭：用户可暂时收起推荐条

## 9. 成功标准

1. **用户体验测试**：用户能清晰感受到"在和顶级安全专家协作"而非"在选择 AI 工具"
2. **角色识别度**：仅通过 UI 配色和布局，用户能区分当前是哪个角色视角
3. **场景匹配度**：智能推荐能准确反映当前最需要关注的安全领域
4. **工作流连续性**：角色切换不打断工作流，数据保持同步
