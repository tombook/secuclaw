# Design: 角色仪表盘面板升级

## 架构

每个角色的 `_render<Role>Dashboard()` 方法返回 2-4 个面板（panel），排列在 CSS Grid 中。

### 面板组件化

每个面板遵循统一结构：
```html
<div class="rd-section">
  <div class="rd-title">📊 标题</div>
  <!-- 面板内容：列表/图表/表格/进度条 -->
</div>
```

### 数据结构

在 `_renderRoleDashboard()` 中使用本地 mock 数据，未来替换为 API 返回。

## CSS 设计

```css
.role-dash {
  padding: 4px 12px;
}
.rd-row {
  display: grid;
  grid-template-columns: 1fr 1fr; /* 默认 2 列 */
  gap: 6px;
}
.rd-section {
  background: #0d1320;
  border: 1px solid #1e293b;
  border-radius: 4px;
  padding: 8px;
}
```

某些角色用 3 列（如指挥官），某些用 1 列（如业务安全）。
