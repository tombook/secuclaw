
# OpenSpec 设计文档: 角色仪表盘UI精简+能力对齐优化
## 版本: v1.0
## 日期: 2026-04-19

---
### 设计原则
1. **一卡一指标**: 每个核心工具面板仅展示1个核心大指标，辅助1个SVG图表/进度条
2. **能力1:1映射**: 每个面板内容严格对应角色light安全能力定义
3. **无冗余无空白**: 三层结构（状态栏→5工具面板→工具事件汇总）完全填充区域
4. **全角色一致性**: 所有角色面板设计规范统一，视觉风格一致

### 布局结构
```
┌───────────────────────────────────────────────┐
│ 状态栏 (角色运行状态 + 核心KPI一行展示)        │
├───────────────────────────────────────────────┤
│  核心工具面板区域 (3+2 grid)                  │
│  ┌──────────┬──────────┬──────────┐           │
│  │ 工具1    │ 工具2    │ 工具3    │           │
│  ├──────────┴──────────┼──────────┤           │
│  │ 工具4               │ 工具5    │           │
│  └─────────────────────┴──────────┘           │
├───────────────────────────────────────────────┤
│ 工具事件汇总 (核心工具状态标签一行展示)        │
└───────────────────────────────────────────────┘
```

### UI规范
1. 核心大指标: 字号24px，字重800，颜色对应状态（绿/橙/红）
2. 辅助图表: 高度不超过24px，宽度自适应，颜色与指标统一
3. 面板标题: 字号12px，字重600，图标+文字，左对齐
4. 面板内边距: 8px，边框1px #2d333d，圆角4px
