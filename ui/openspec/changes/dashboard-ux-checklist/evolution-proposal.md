## 1. 角色仪表盘UX检查清单
在更新角色仪表盘时，必须执行以下4步检查：

### Step 1: 空白区域检查
- 确保5个核心工具的仪表盘完全填充右侧区域，无空白
- 每个面板包含：核心指标 + 可视化图表 + 详细内容 + 任务提示

### Step 2: 明确任务提示检查
- 每个面板必须添加明确的待处理任务提示
- 使用颜色区分优先级：🔴红色（紧急/待审批）、🟡黄色（需优化/有差距）、🟢绿色（正常/达标）
- 任务提示格式：`<span class="rd-dot" style="background:颜色"></span><span class="rd-text" style="flex:1;font-size:10px">任务内容</span>`

### Step 3: 工具面板集成检查
- 在使用面板的文件中必须导入工具面板组件
- 必须添加`open-tool-panel`事件监听器
- 必须添加状态管理工具面板的打开/关闭
- 必须在render方法中渲染工具面板组件

### Step 4: 底部工具列表映射检查
- 确保工具列表正确映射到角色的5个核心工具
- 确保点击工具或面板可以正常打开工具面板
- 确保工具面板支持3种模式：右侧滑出/居中模态/全屏覆盖
